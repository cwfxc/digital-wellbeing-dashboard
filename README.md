# Digital Wellbeing Dashboard

Учебный проект по аналитике: исследование связи между цифровым поведением и показателями ментального здоровья на синтетических данных. Подготовка данных в Python, визуализация в Power BI.

## Инструменты

- **Python** (pandas, numpy) — подготовка и преобразование данных
- **Power BI** — построение интерактивного дашборда
- **DAX** — пользовательские меры для KPI-карточек
- **Power Query** — загрузка и трансформация данных

## Данные

Источник: [Mental Health and Digital Behavior 2020-2024](https://www.kaggle.com/datasets/atharvasoundankar/mental-health-and-digital-behavior-20202024) на Kaggle.

⚠️ **Важно:** данные **симулированные** (сгенерированы программно для задач машинного обучения). Это означает, что выводы из анализа применимы только к самому датасету и не отражают реальные закономерности.

В датасете 9 числовых показателей:
- Цифровое поведение: экранное время, переключения приложений, время в соцсетях, количество уведомлений
- Самооценка состояния: тревожность, концентрация, настроение, часы сна, интегральный балл цифрового благополучия

## Задачи проекта

1. Понять, какие цифровые привычки сильнее всего связаны с уровнем тревожности и концентрации
2. Выделить группу с самыми высокими показателями тревожности и описать её профиль
3. Отработать навыки анализа данных в Python и построения дашборда в Power BI

## Что сделано

**Подготовка данных (Python):**
- Загрузка и осмотр исходного CSV
- Расчёт корреляций Пирсона между цифровыми привычками и показателями состояния
- Выделение топ-10% наблюдений с наивысшим уровнем тревожности через квантиль
- Сравнение средних значений у топ-10% и по всей выборке
- Экспорт результатов в три отдельных CSV для Power BI

**Дашборд в Power BI:**
- Двухстраничная структура: общий анализ + детальный профиль группы риска
- KPI-карточки с DAX-мерами (формат "8,45 / 10")
- Барчарт корреляций с двумя показателями (тревога/концентрация)
- Точечный график с категориальной разбивкой
- Кольцевая диаграмма распределения категорий тревожности
- Карточки с отклонениями от среднего (+/-%)

## Что показал анализ

**В этом датасете:**
- Сильнее всего с тревожностью связаны время в соцсетях (r ≈ 0.31) и количество уведомлений (r ≈ 0.30)
- Те же факторы отрицательно связаны с концентрацией
- 86.6% наблюдений попадают в категорию "высокая тревожность"
- У топ-10% с самой высокой тревожностью соцсети и уведомления на 10-11% выше среднего, а интегральный балл самочувствия на 9.6% ниже

## Ограничения проекта

- Данные синтетические — выводы нельзя переносить на реальное поведение людей
- Возраст и пол респондентов не указаны в датасете
- Цель проекта — демонстрация технических навыков, а не научное исследование
